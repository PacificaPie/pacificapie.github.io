A skill is a small, versioned playbook an AI agent can load: trigger conditions, an execution order, and hard stops — with the knowledge it needs living in one authoritative document it points to. Whenever a workflow of mine starts repeating, I compile it into one. These are the seven I can show. skill 是 agent 可加载的小型剧本：触发条件、执行序、硬中止， 它依赖的知识正文放在它指向的唯一权威文档里。 一个工作流开始重复出现，我就把它固化成一个 skill。这里是可以公开的 7 个。

Thin shell薄皮原则

A skill carries triggers, order, and stops — never a second copy of the knowledge. One source of truth; everything else is a pointer. skill 只带触发、顺序与中止，不复制知识正文——同一知识只在一处有正文，其余都是指针。

Write-back经验回写

Every real use writes its lessons back into the skill's own knowledge base. The tool gets sharper the more it runs. 每次真实使用都把教训写回 skill 自己的判例与知识库，工具越用越准。

Knowing when not to orchestrate知道何时不编排

Independent playbooks get no orchestration layer. Structure is added when a trigger signal appears — not before. 互不依赖的剧本不上编排层；结构等触发信号出现才加，绝不提前。

Product line产品线 a product and its research harness, built as a pair 产品与它的调研 harness，成对出现
tripalarm product产品

TripAlarm — say where, when, and your budget; a server watches prices year-round and pushes the moment they hit (ntfy / Feishu / Telegram). The skill is the product's front door. 旅行闹钟——说出目的地、时间和预算，server 全年盯价， 到价即主动推送（ntfy / 飞书 / Telegram）。skill 本身就是产品入口。

competitor-research research harness调研 harness

Exhaustive competitor research for TripAlarm: bilingual sweeps across apps, web tools, and AI agents, with live screenshots — and a registry ledger that refuses to stop until the search is truly exhausted. 为 TripAlarm 做穷尽式竞品调研：中英双语横扫 App / 网页 / AI agent 三种形态并实地截图； registry 登记簿保证搜到「搜无可搜」才允许收尾。

Diagram twins配图双子星 one export craft, two visual grammars for two audiences 同一套导出工艺，面向两种场合的两套语法
glass-diagram workflow工作流

Business-report diagrams in a unified glass style: HTML skeleton → design system → headless-Chrome self-check → 2× PNG. Consistency is the product — the same object gets the same icon in every chart. 蓝白玻璃质感的汇报配图流水线：HTML 骨干 → 设计系统渲染 → 无头 Chrome 截图自检 → 2 倍 PNG。 一致性即产品——同一业务物件在所有图里长同一个样。

academic-diagram workflow工作流

Paper-style layered architecture figures: a shape-semantics grammar (cylinder = store, pill = AI capability…), figure-in-figure nesting, and one hard rule — arrows are flows, no exceptions. 学术风分层架构图：形状语义语法（圆柱=存储、胶囊=AI 能力…）、 图套图工序（子图独立渲染再嵌回父图），以及一条铁律——箭头即流向，没有例外。

Personal systems个人系统 the material layer of an agent that runs every day 一个每天在跑的 agent 的素材层
career-archive system系统

A private experience archive in three strict layers: capture (append-only truth log) → organize (direction-neutral distillation) → tailor (reserved). Two human review gates — nothing enters the archive without sign-off. It is the material layer of my job-search agent. 私有经历库，三层分工：捕获（只追加的流水真相源）→ 整理（方向中立的研磨）→ 裁剪（预留）。 两道人工审核闸，不经确认不入库；它是我求职 agent 的素材层。
